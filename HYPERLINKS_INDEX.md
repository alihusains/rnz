## üîç Feature Overview

### What It Does
- Detects `<hyperlink-name>` patterns in description text
- Renders as blue underlined interactive links
- Tapping navigates to related content
- Preserves all user preferences
- Supports nested hyperlinks

### How It Works
1. App detects `<text>` in description
2. Renders without brackets as blue link
3. User taps link
4. App queries database for hyperlink content
5. New screen opens with results
6. User can tap back or explore nested hyperlinks

### Key Technologies
- **Regex**: Pattern matching for hyperlinks
- **RichText**: Inline hyperlink rendering
- **SQLite**: Database queries for hyperlink content
- **Flutter Navigation**: Screen transitions
- **SharedPreferences**: Preference persistence
